# X²-Gaussian (X2GS) 最优超参配置

## 概述

X²-Gaussian 是基于 K-Planes 的特征增强技术，用于 CT 重建中的 3D Gaussian Splatting。该技术通过学习空间变化的特征调制来增强高斯点的表达能力。

## 最优超参配置

| 参数 | 最优值 | 说明 |
|------|--------|------|
| K-Planes 分辨率 | 64×64 | 每个平面的空间分辨率 |
| 特征维度 | 32 | 每个平面的特征通道数 |
| 平面数量 | 3 | XY, XZ, YZ 三个正交平面 |
| 总特征维度 | 96 | 3 planes × 32 features |
| MLP 层数 | 3 | 解码器网络深度 |
| MLP 隐藏维度 | 128 | 解码器隐藏层宽度 |
| TV Lambda | 0.002 | Total Variation 正则化权重 |
| 激活函数 | Tanh | 输出约束在 [-1, 1] |
| 调制公式 | 1.0 + 0.5 × offset | 最终调制范围 ±50% |

## 关键设计决策

1. **Tanh 激活约束**：确保调制系数在有界范围内，防止训练不稳定
2. **温和调制范围 (±50%)**：避免过度调制导致的伪影
3. **TV 正则化**：促进 K-Planes 特征的空间平滑性，减少噪声

## 核心洞察

### 1. X²-Gaussian (K-Planes) 是最稳定的增益技术

| 场景 | X 单独效果 | 最佳组合 |
|------|-----------|----------|
| Foot-3 | -0.09 dB | IX (+0.23) |
| Chest-6 | +0.01 dB | XF (+0.09) |
| Abdomen-9 | **+0.17 dB** | X (+0.17) |

**结论**: X 在视角越多的场景效果越好，且是多数最佳组合的核心成分

### 2. Init-PCD 存在场景适应性问题

| 场景 | I 单独效果 |
|------|-----------|
| Foot-3 | -0.06 dB |
| Chest-6 | -0.13 dB |
| Abdomen-9 | -0.30 dB |

**结论**: Init-PCD 在密集场景反而有负面影响，需要进一步调研原因

### 3. 技术组合不是简单叠加

- Foot-3: IX 效果远好于 I + X 单独效果之和
- Chest-6: IXFB (全技术) 反而低于 baseline
- 存在技术间的协同和冲突效应

### 4. Bino 双目一致性表现不佳

在所有三个场景中，B 技术均为负增益或接近零，可能原因：
- 参数未针对 CT 场景优化
- 与其他技术存在冲突

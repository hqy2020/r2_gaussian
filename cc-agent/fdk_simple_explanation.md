# FDK 算法：用"影子"反推物体

## 🎮 互动演示：猜猜盒子里是什么？

### 游戏规则
- 盒子里有个神秘物体（您看不到）
- 您只能看到 3 个方向照射的"影子"
- 根据影子，猜出物体是什么

---

## 📸 案例 1：猜球还是方块？

```
从上往下照：          从左往右照：         从斜 45° 照：
     💡                   💡                  💡
      ↓                    ↓                   ↓
   [盒子]                [盒子]               [盒子]
      ↓                    ↓                   ↓
    ⚫⚫                  ⚫⚫                 ⚫⚫
   ⚫⚫⚫               ⚫⚫⚫                ⚫⚫
    ⚫⚫                  ⚫⚫                 ⚫⚫

  (圆形影子)            (圆形影子)           (圆形影子)
```

**您的猜测**：里面是一个 **球** 🏀

**为什么**？
- 所有角度看都是圆的
- 只有球才会这样

**这就是 FDK 算法的核心思想！**

## 🦴 案例 2：重建 Foot（脚部）CT

### 输入：3 个角度的"影子"（投影图）

```
角度 1 (0°):           角度 2 (120°):        角度 3 (240°):
正面照脚               从右侧照              从左侧照

投影图灰度：
⬜⬜⬜⬜⬜            ⬜⬜⬜⬜              ⬜⬜⬜⬜
⬜⬜⬛⬛⬜  ← 骨骼    ⬜⬛⬜⬜              ⬜⬛⬜⬜
⬜⬛⬛⬛⬜    阴影    ⬛⬛⬜⬜              ⬜⬛⬛⬜
⬜⬜⬛⬛⬜            ⬜⬛⬜⬜              ⬜⬜⬛⬜
⬜⬜⬜⬜⬜            ⬜⬜⬜⬜              ⬜⬜⬜⬜

(⬛ = 暗，X射线被挡)  (⬛ = 暗)            (⬛ = 暗)
(⬜ = 亮，X射线穿过)  (⬜ = 亮)            (⬜ = 亮)
```

### FDK 算法的处理过程

#### 步骤 1：初始化 3D 空间

```python
# 创建空的 3D 网格（脚部区域）
vol = np.zeros((512, 512, 512))

# 这是一个巨大的 3D 方格纸
# 每个小格子（体素）存储一个密度值
# 初始全部为 0
```

#### 步骤 2：反投影（核心！）

**反投影 = 把影子"涂抹回"3D 空间**

```
角度 1 的处理：
━━━━━━━━━  ← 这是投影的一条"暗线"（骨骼）
   ↓↓↓     把这条暗线沿着射线方向"涂回去"
━━━━━━━━━
━━━━━━━━━  ← 沿着这条射线，所有体素都加上一些密度
━━━━━━━━━

结果：
vol[x, y, z] += 0.8  (沿着暗线的所有体素)
vol[x, y, z] += 0.1  (沿着亮线的所有体素)
```

**关键**：每个角度都这样"涂抹"一次

```
只用角度 1：              加上角度 2：             再加角度 3：
   ━━━                     ━━━                    ╱━━━╲
   ━━━  (模糊)           ╱━━━╲ (交叉处清晰)    ╱ 🦴 ╲ (骨骼清晰！)
   ━━━                   ╲━━━╱                  ╲━━━╱
```

**为什么交叉处密度高**？
- 骨骼位置：3 个角度都投射了"暗"值 → 累加 → 密度高 (0.8+0.8+0.8)/3 = **0.8**
- 空气位置：3 个角度都投射了"亮"值 → 累加 → 密度低 (0.1+0.1+0.1)/3 = **0.1**

#### 步骤 3：平均

```python
vol = vol / 3  # 除以角度数量
# 最终结果：
vol[骨骼位置] = 0.5   # 高密度
vol[空气位置] = 0.02  # 低密度
vol[软组织]   = 0.1   # 中密度
```

## 🎨 可视化：FDK 的"交叉定位"原理

```
从 3 个角度照射一个骨骼：

角度 1 (━):
━━━━━━━━━━━━━━━  ← 这条射线经过骨骼，投影暗
         ↓↓↓
      [3D空间]


角度 2 (╱):
        ╱╱╱
       ╱╱╱   ← 这条射线也经过骨骼，投影暗
      ╱╱╱
     [3D空间]


角度 3 (╲):
     ╲╲╲
      ╲╲╲   ← 这条射线同样经过骨骼，投影暗
       ╲╲╲
    [3D空间]


3 个射线的交点：
       ╱━━╲
      ╱ 🦴 ╲  ← 唯一的交点 = 骨骼的真实位置！
     ╱━━━━╲    3 条暗射线都经过这里
                → 密度累加 → 高密度 ✅
```

**非交叉点**（空气）：
- 只有 1 条或 0 条射线经过
- 密度累加少
- 最终密度低 ✅

## 💡 总结：FDK 的本质

### 核心思想
> **多个角度的投影射线，交点处就是真实物体**

### 类比

| 类比 | FDK 对应 |
|------|----------|
| 手电筒 | X 射线源 |
| 影子 | 投影图 |
| 影子的暗处 | X 射线被挡住 → 高密度 |
| 影子的亮处 | X 射线穿过 → 低密度 |
| 从多个角度看影子 | 多个投影角度 |
| 推测物体形状 | 重建 3D 密度 |

### 为什么需要多个角度？

```
只有 1 个角度：
━━━━━  ← 无法判断是球还是柱子
  ？
━━━━━

有 3 个角度：
━━━, ╱╱╱, ╲╲╲  ← 交叉点确定形状
      ⚽ 是球！
```

## 🔢 数学直觉（可选）

如果您想了解一点数学（可跳过）：

```
正向投影（已知物体 → 计算影子）：
projection[angle] = Σ density[x,y,z]  沿射线积分

逆向重建（已知影子 → 推测物体）：
density[x,y,z] = Σ projection[angle]  从所有角度累加
```

**FDK 算法 = 巧妙的逆向重建方法**

## 🎯 回到密度加权采样

现在您明白了：

1. **FDK 输入**：3 个角度的投影图（影子）
2. **FDK 处理**：反投影 + 交叉累加
3. **FDK 输出**：3D 密度网格 `vol[x,y,z]`
   - 骨骼位置：`vol = 0.5`（高密度）
   - 空气位置：`vol = 0.02`（低密度）

4. **密度加权采样**：
   ```python
   # 从 vol 中按密度概率采样
   probs = densities / densities.sum()
   # 骨骼（0.5）被采样的概率 = 0.5 / Σ = 高
   # 空气（0.02）被采样的概率 = 0.02 / Σ = 低
   ```

**整个流程是物理驱动的**，没有任何人工标注！

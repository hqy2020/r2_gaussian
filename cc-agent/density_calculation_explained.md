# 密度值的计算原理：从X射线投影到密度重建

**回答问题：密度加权采样中的"密度"是根据什么算出来高还是低的？**

---

## 🎯 简短回答

**密度值来源于 FDK 重建算法，它根据 X 射线投影的衰减程度反推出每个体素的密度。**

- **骨骼密度高** = X 射线穿过时衰减多 = 投影图像上显示暗
- **空气密度低** = X 射线穿过时衰减少 = 投影图像上显示亮

---

## 📖 完整解释：从物理原理到数值计算

### 步骤 1：X 射线投影的物理原理

#### Beer-Lambert 定律

当 X 射线穿过物体时，强度会被吸收（衰减）：

```
I_out = I_in × exp(-∫ μ(s) ds)

其中：
- I_in:  入射 X 射线强度
- I_out: 出射 X 射线强度
- μ(s):  路径上每个点的线性衰减系数（linear attenuation coefficient）
- ∫ ds:  X 射线路径积分
```

**关键点**：`μ(s)` 就是我们说的"密度"！

#### 不同组织的衰减系数 μ（物理测量值）

| 组织类型 | 衰减系数 μ (cm⁻¹) | 说明 |
|---------|-------------------|------|
| **空气** | ~0.0002 | 几乎不吸收 X 射线 |
| **水/软组织** | ~0.20 | 中等吸收 |
| **肌肉** | ~0.22 | 略高于水 |
| **骨骼** | ~0.50 - 1.20 | 强吸收（含钙质） |

**为什么骨骼密度高？**
- 骨骼含有大量钙（Ca）和磷（P）元素
- 钙的原子序数高（Z=20），对 X 射线吸收强
- 因此 μ 值大 → "密度"高

**为什么空气密度低？**
- 空气主要是氮（N₂）和氧（O₂）
- 气体稀疏，几乎不吸收 X 射线
- 因此 μ 值接近 0 → "密度"低

---

### 步骤 2：投影图像的形成

X 射线探测器测量的是衰减后的强度：

```python
# 投影值 p（对数化的衰减）
p = -log(I_out / I_in) = ∫ μ(s) ds

# 这就是 CT 扫描仪记录的原始数据（投影图 projs_train）
```

**投影图的灰度值**：
- **暗（高值）** → X 射线被大量吸收 → 路径上有高密度组织（骨骼）
- **亮（低值）** → X 射线几乎无衰减 → 路径上是低密度组织（空气）

---

### 步骤 3：FDK 重建算法

FDK (Feldkamp-Davis-Kress) 算法是圆锥束 CT 的标准重建方法，它的任务是：

**输入**：多个角度的投影图 `p_θ(u, v)`（3 张、6 张或 9 张）
**输出**：3D 体素网格 `vol(x, y, z)`，每个体素存储密度 μ

#### FDK 算法数学公式

```
重建过程（简化版）：

1. 滤波：对每个投影应用 ramp filter（高频增强）
   p_filtered = p * h_ramp

2. 反投影：从所有角度将滤波后的投影"投射回"3D空间
   μ(x,y,z) = Σ_θ [ w(θ) × p_filtered_θ(u(x,y,z,θ), v(x,y,z,θ)) ]

其中：
- θ: 投影角度（如 0°, 120°, 240° 对于 3 views）
- w(θ): 几何权重因子
- (u,v): 将3D点投影到探测器的2D坐标
```

**核心思想**：
- 每个投影告诉我们"某条射线路径上密度的总和"
- FDK 通过反向投影，将这些"总和"分配回3D空间的每个体素
- 多个角度的投影相交，逐步重建出每个体素的密度

#### R²-Gaussian 代码实现

```python
# r2_gaussian/utils/ct_utils.py
def recon_volume(projs, angles, geo, method='fdk'):
    """
    projs: [N_views, H, W] 投影图像
    angles: [N_views] 投影角度（弧度）
    geo: TIGRE 几何参数
    method: 'fdk' 使用 FDK 算法

    返回: vol [nx, ny, nz] 3D密度体素网格
    """
    # 调用 TIGRE 库的 FDK 实现
    vol = algs.fdk(projs, geo, angles)

    # vol 中的每个值就是该位置的密度 μ
    return vol
```

---

### 步骤 4：密度值的数值范围

FDK 重建后的 `vol` 数组：

```python
vol = recon_volume(projs_train, angles_train, geo, 'fdk')
# vol.shape: (512, 512, 512)  示例尺寸
# vol 的数值含义：

vol[i, j, k] = μ_ijk  # 体素 (i,j,k) 的线性衰减系数

# 典型范围（归一化后）：
# - 空气:     0.000 - 0.050
# - 软组织:   0.050 - 0.150
# - 骨骼:     0.250 - 1.000+
```

#### 与 CT 中的 HU 值对应

在临床 CT 中，密度通常用 **Hounsfield Unit (HU)** 表示：

```
HU = 1000 × (μ - μ_water) / μ_water

典型 HU 值：
- 空气:        -1000 HU  →  μ ≈ 0
- 水:             0 HU  →  μ = μ_water (基准)
- 软组织:    30-70 HU  →  μ ≈ 1.03 × μ_water
- 骨骼:    300-1000 HU  →  μ ≈ 1.3-2.0 × μ_water
```

**R²-Gaussian 中的密度值**是 μ 的归一化版本（未经 HU 转换）。

## 🔬 实例演示：Foot-3 views 的密度计算

### 输入数据

```python
# 3 张投影图（3个角度：0°, 120°, 240°）
projs_train.shape = (3, 512, 512)

# 投影图的灰度值
projs_train[0, 256, 256] = 0.85  # 中心像素，X射线被大量吸收（骨骼）
projs_train[0, 100, 100] = 0.05  # 边缘像素，X射线几乎无衰减（空气）
```

### FDK 重建过程（简化）

```python
# 1. 滤波增强
projs_filtered = apply_ramp_filter(projs_train)

# 2. 反投影
vol = np.zeros((512, 512, 512))

for angle_idx, angle in enumerate([0°, 120°, 240°]):
    # 将过滤后的投影反投影到3D空间
    for x, y, z in all_voxels:
        # 计算体素 (x,y,z) 投影到探测器的坐标 (u,v)
        u, v = project_to_detector(x, y, z, angle, geo)

        # 累加贡献
        vol[x, y, z] += projs_filtered[angle_idx, u, v] * weight(angle, x, y, z)

# 3. 归一化
vol = vol / num_angles  # 平均
```

### 输出密度分布

```python
# 重建后的体素网格
vol[256, 256, 256] = 0.652  # 脚部中心（骨骼），密度高
vol[100, 100, 100] = 0.018  # 脚部外围（空气），密度低
vol[200, 256, 300] = 0.082  # 脚部软组织，密度中等

# 这些值就是后续采样策略中使用的"密度"！
```

## 🎯 回到密度加权采样

现在我们知道了密度是怎么来的，密度加权采样就很清楚了：

### 采样概率计算

```python
# 1. FDK 重建得到 3D 密度网格
vol = recon_volume(projs, angles, geo, 'fdk')
# vol[x,y,z] = 该体素的线性衰减系数 μ

# 2. 过滤低密度（空气）
valid_indices = np.argwhere(vol > 0.05)  # 只保留密度 > 0.05 的体素
densities = vol[valid_indices[:, 0],
                valid_indices[:, 1],
                valid_indices[:, 2]]

# 3. 密度加权采样概率
probs = densities / densities.sum()  # 归一化为概率分布

# 采样概率举例：
# - 骨骼体素（μ=0.5）: prob = 0.5 / Σμ  （高概率）
# - 软组织（μ=0.1）:   prob = 0.1 / Σμ  （中概率）
# - 空气边界（μ=0.06）: prob = 0.06 / Σμ （低概率，已过滤大部分）

# 4. 按概率采样
sampled_idx = np.random.choice(len(valid_indices), 50000,
                               replace=False, p=probs)
```

### 为什么有效？

因为 **FDK 重建的密度值 = 物理真实的 X 射线吸收能力**：
- 骨骼真的含钙，真的强吸收 X 射线 → μ 大
- 空气真的稀薄，真的不吸收 X 射线 → μ 小

密度加权采样 = **根据物理真实性分配采样权重**

## 💡 关键洞察

**密度不是"算"出来的，而是"测"出来的！**

1. **物理测量**：X 射线穿过身体，探测器测量衰减
2. **数学反演**：FDK 算法从投影反推每个体素的密度
3. **采样策略**：密度加权采样利用这个物理真实的密度值

**密度值 = 组织的物理属性（线性衰减系数 μ）的数值表示**

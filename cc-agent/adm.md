# ADM 模块文档

## 概述

ADM (Alternating Direction Method) 是一种用于CT重建优化的模块，通过交替方向优化来提升重建质量。

## 超参数配置

### lambda_tv (Total Variation 权重)

**推荐值**: `lambda_tv = 0.001` 或 `lambda_tv = 0.002`

#### 消融实验结果 (2025-12-05)

实验路径: `output/ablation_adm_lambda_tv_2025_12_05_11_48/`

数据集配置:
- 数据: `data/369/foot_50_3views.pickle`
- SPS点云: `data/density-369/init_foot_50_3views.npy` (3k点)
- GAR邻居点数: 5

| lambda_tv | PSNR3D | SSIM3D | PSNR2D | SSIM2D | 备注 |
|-----------|--------|--------|--------|--------|------|
| 0.0005 | 23.078 | 0.720 | 28.703 | 0.899 | |
| **0.001** | **23.079** | 0.720 | 28.850 | **0.900** | PSNR3D最优 |
| **0.002** | 23.075 | 0.720 | **29.026** | **0.900** | PSNR2D最优 |
| 0.005 | 23.053 | 0.720 | 28.872 | 0.899 | |
| 0.01 | 23.073 | 0.720 | 28.660 | 0.899 | |

#### 结论

1. **3D重建质量优先**: 选择 `lambda_tv = 0.001`
   - PSNR3D达到最高 (23.079)

2. **2D投影质量优先**: 选择 `lambda_tv = 0.002`
   - PSNR2D达到最高 (29.026)

3. **综合推荐**: `lambda_tv = 0.001`
   - 3D和2D指标都接近最优
   - SSIM2D达到最高 (0.900)

4. **不推荐的值**:
   - `lambda_tv >= 0.005`: TV正则化过强，影响细节保留
   - `lambda_tv <= 0.0005`: TV正则化过弱，去噪效果不明显

## 使用示例

```bash
# 推荐配置
python train.py --lambda_tv 0.001 ...

# 如果需要更强的2D投影质量
python train.py --lambda_tv 0.002 ...
```

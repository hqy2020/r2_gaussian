# Init-PCD 超参数搜索结论

## 核心结论

**低阈值 + 低权重 + 极少点数** 在稀疏场景效果最好。

### 🏆 最佳配置（2024-12-04 更新）
- `n_points`: **3,000**（极少点数，训练后增长至 ~4.8k）
- `density_thresh`: 0.03（低阈值）
- `density_rescale`: 0.10（低权重）

### 性能提升
- **2D PSNR**: 29.80 dB（比 baseline **+1.31 dB**）
- **2D SSIM**: 0.9073（比 baseline **+0.68%**）

## SPS n_points 细粒度消融实验（2024-12-04）

### 实验背景
在初步发现"少点数效果更好"后，对 1k-25k 范围进行细粒度搜索，找到最优 n_points。

### 完整结果

| 初始点数 | 2D PSNR | 2D SSIM | vs Baseline | 最终点数 | 增长倍数 |
|---------|---------|---------|-------------|---------|---------|
| 1k | 28.88 | 0.9045 | +0.40 dB | 3,471 | 3.47x |
| 2k | 29.45 | 0.9032 | +0.96 dB | 4,205 | 2.10x |
| **3k** 🏆 | **29.80** | **0.9073** | **+1.31 dB** | 4,770 | 1.59x |
| 5k | 29.45 | 0.9023 | +0.96 dB | 6,345 | 1.27x |
| 8k | 29.36 | 0.9000 | +0.87 dB | 9,075 | 1.13x |
| 10k | 29.28 | 0.8996 | +0.79 dB | 10,991 | 1.10x |
| 15k | 29.02 | 0.8991 | +0.53 dB | 15,790 | 1.05x |
| 20k | 28.98 | 0.8968 | +0.49 dB | 20,551 | 1.03x |
| 25k | 28.90 | 0.9039 | +0.41 dB | 25,354 | 1.01x |

**Baseline**: 50k 随机采样，PSNR 28.4873, SSIM 0.9005

### 关键发现

1. **最优点数 = 3k**：PSNR 峰值在 3k，低于或高于都会下降
2. **~4k 即可满足高质量需求**：3k 初始点云训练后增长至 4.8k，达到最佳效果
3. **点云增长规律**：
   - 初始点云越小，训练中增长越多（1k → 3.47x）
   - 初始点云越大，几乎不增长（25k+ → ~1.0x）
4. **效率优势**：3k 初始点云比 50k baseline 减少 94% 的初始计算量

### 推荐配置

```yaml
# 3views 极稀疏场景最佳配置
n_points: 3000          # 初始点云数量
density_thresh: 0.03    # 密度阈值
density_rescale: 0.10   # 密度权重

# 预期效果
# - 训练后点云数: ~4,800
# - PSNR 提升: +1.31 dB
# - SSIM 提升: +0.68%
```

## 场景适应性

| 场景 | Init-PCD 效果 | 说明 |
|------|--------------|------|
| 3views（极稀疏） | ✅ 有效 | 最佳配置可提升 +0.276 dB |
| 9views（密集） | ❌ 有害 | 所有配置均低于 baseline |

## 规律总结

1. **点数越少越好**：25k > 50k > 75k
2. **阈值越低越好**：0.03 > 0.05 > 0.08
3. **权重越低越好**：0.10 效果最佳
4. **场景限制**：仅适用于极稀疏场景（3views），密集场景（9views）会造成性能下降
